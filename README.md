Demo of Ajax with a ToDo List
=========================

**jQuery and Ajax**

This is my solution for a ToDo List which follows the Ryan Bates
Railscast #136 revised on jQuery and  Ajax.  I also implented the jQuery code using
CoffeeScript.

Requirements
-------------------

 - Ruby v. 2.2.2
 - Rails v. 4.2.1
 - jQuery-rails v. 4.1.0
 - Postgresql
